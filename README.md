# CommerceHub

CommerceHub is a full-stack e-commerce web application developed using Java Spring Boot for the backend and HTML, CSS, JavaScript, and Bootstrap for the frontend. The application allows users to browse products, manage their cart and wishlist, save delivery addresses, and place orders through a secure authentication system.

## Features

- User Registration and Login
- Product Listing
- Product Search
- Add to Cart
- Wishlist Management
- Buy Now
- Address Management
- Order Placement
- Order History
- Responsive User Interface

## Application Workflow

1. User Registration and Login
2. JWT Authentication
3. Browse Products
4. Add Products to Cart or Wishlist
5. Save Delivery Address
6. Buy Products
7. Place Orders
8. View Order History
